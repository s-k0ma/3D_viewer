# 3D_viewer
Десктопное приложение для отрисовки каркасных моделей 3D объектов

Реализовано с помощью Qt, OpenGL и принципов ООП

## Сборка
Сборка осуществляется командой ``make install``, перед запуском убедится о наличии необходимых зависимостей  
### Тестовый запуск  
Для пробного запуска предусмотрена возможность собрать Docker контейнер и запустить приложение из него  
``docker compose up --build``  
Перед запуском необходимо выполнить команду ``xhost +si:localuser:$(whoami)`` для обеспечения подключения к монитору изнутри контейнера  
После работы необходимо отключить настройку командой ``xhost -si:localuser:$(whoami)``
